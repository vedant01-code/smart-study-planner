# 📚 Smart Study Planner
A modern, intelligent study planner web app that helps students manage subjects, track progress, and plan daily study schedules based on exam dates and topic priorities.

---

## 🚀 Live Demo
👉 https://vedant01-code.github.io/smart-study-planner/

---

## ✨ Features

### 📅 Smart Planning
- Auto-generates daily study plan based on exam dates
- Prioritizes important topics first

### 📊 Progress Dashboard
- Tracks completion percentage for each subject
- Visual progress bars

### 🧠 Intelligent System
- Priority-based topic learning
- Adaptive study suggestions

### 🎮 Gamification
- XP system for completed topics
- Streak tracking (study consistency)

### 📖 Subject Management
- Add subjects easily (DBMS, OS, Math, CN, Python, AI)
- Delete subjects
- Track topic completion

### ⏱ Study Timer
- Pomodoro-style study timer (25 min default)

### 💾 Local Storage
- Saves all data in browser automatically
- No login required

## ⚙️ How to Run Locally

1. Download or clone this repository
2. Open `index.html` in your browser
3. Start adding subjects and planning your studies

---

## 📌 How It Works

1. Add a subject with exam date  
2. App assigns topics automatically  
3. System creates a daily study plan  
4. Track progress via dashboard  
5. Earn XP for completing topics
